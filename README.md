# Server Sessions

Both of these apps include:

- A public website area
- User Signup
- User Login & Logout
- A protected members-only page
- Server side sessions (in-memory)
- A mocked User DB (a simple array)

## `server-sessions-1`

Differences:
- Uses inline HTML (no templates)
- Extra logging
- .`http` files with examples

## `server-session-2`

Differences:
- Uses EJS templates
- Additional checks on User sign-up. 
- Includes a default user
